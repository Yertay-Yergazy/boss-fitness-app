# BOSS FITNESS Design System

Complete visual system for BOSS FITNESS — a premium gym management app with dark theme, gold accents, and modern UI components.

## Quick Start

1. Read `readme.md` for visual foundations and product overview
2. Browse `assets/` for logos and trainer photos
3. Check `ui_kits/mobile_app/index.html` for interactive prototype
4. Use components from `components/` to build new screens
5. Reference CSS tokens in `styles.css` for consistent styling

## Key Colors

- **Dark bg:** `#0A0A0A`
- **Gold primary:** `#C9A227`
- **Gold light:** `#F5D77A`
- **Red accent:** `#C81E1E`

## Fonts

- **Display:** Anton (bold, all caps)
- **Body:** Montserrat (multiple weights)

## Components Available

- Button (primary/secondary)
- Card (dark with gold border)
- Badge (status indicators)
- ProgressBar (subscription tracking)
- Avatar (user profile)
- KeyBadge (locker selection)
- SubscriptionCard (membership display)
- TrainerCard (trainer profiles)

## Mobile App Features

- QR check-in flow
- Locker key selection
- Trainer booking
- Subscription management
- Visit history
- Profile & settings

Use this system to design new gym features, additional screens, or extend the existing app.
