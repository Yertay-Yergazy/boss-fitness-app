# Home — `/`

**Тип:** Server Component. **Низ экрана:** `<TabBar active="home">`.

## Данные (загружаются на сервере)

- `GET /api/profile` → имя, инициалы
- `GET /api/subscription` → карточка абонемента
- `GET /api/sessions?upcoming=true&limit=1` → ближайшая запись (для блока "Ближайшая запись"); если нет — блок не показываем

## Layout (сверху вниз)

1. Хедер: инициалы-аватар + "Добро пожаловать, ИМЯ" + иконка-колокольчик (декоративная, без действия)
2. `<SubscriptionCard>` — текущий абонемент, с QR-декорацией
3. Большая CTA-кнопка "ЧЕК-ИН" → `<Link href="/checkin">` (золотой градиент, без onClick — чистая навигация)
4. Сетка 2×2 "Быстрые действия": Записаться к тренеру → `/trainers`, Мои занятия → `/history`, Тренеры → `/trainers`, Купить абонемент → `/plans`. Каждая — `<Link>`.
5. "Ближайшая запись" — карточка тренера → `<Link href="/trainers/[id]">`

## Примечания

Чисто презентационный экран без локального состояния → Server Component, переходы только через `<Link>`.
