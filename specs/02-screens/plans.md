# Абонементы — `/plans`

**Тип:** Server Component. **Верх:** `<BackButton>` → предыдущая страница (`/` либо `/profile`, обе ведут сюда). **Низ:** нет TabBar (см. `00-overview.md` п.4 архитектурных решений — отступление от прототипа, где этот экран по ошибке подсвечивал таб "Главная").

## Данные

`GET /api/plans` → `[{ id, name, icon, price, description }]`

## Layout

Заголовок "АБОНЕМЕНТЫ". Карточка на каждый тариф (12 занятий / безлимит / разовое), кнопка "КУПИТЬ" → `<Link href="/">` (mock-этап: покупка не оформляется, просто возвращает на главную, как в прототипе).
