# История — `/history`

**Тип:** Server Component. **Низ:** `<TabBar active="sessions">`.

## Данные

`GET /api/sessions` → список `{ date, time, lockerNumber, type: 'self'|'group'|'individual', trainerName? }`, отсортированы от новых к старым. Плюс счётчик посещений за месяц (можно вычислить на бекенде и вернуть отдельным полем `visitsThisMonth`).

## Layout

Заголовок "ИСТОРИЯ" + "Посещений в этом месяце: N". Список карточек: иконка-чекмарк, дата·время, имя тренера или "Самостоятельная тренировка", тип занятия, номер ключа.
