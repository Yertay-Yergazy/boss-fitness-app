# Успех — `/checkin/success?key=N`

**Тип:** Client Component — таймер обратного отсчёта. **Низ:** нет TabBar/BackButton, только кнопка "НА ГЛАВНУЮ".

## Данные

`key` читается из `useSearchParams()`. Если отсутствует — fallback на `14` (как в прототипе) или редирект на `/`.

## Состояние

`countdown: number` (старт 3), `setInterval` каждую секунду, при достижении 0 → `router.push('/')`.

## Layout

Анимированный чекмарк (pulse), "ЗАНЯТИЕ ПОДТВЕРЖДЕНО", карточка с номером ключа, время, текст "Переход на главную через N сек...", кнопка "НА ГЛАВНУЮ" (`router.push('/')`).
