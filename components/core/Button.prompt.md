A gold/dark-themed action button for the BOSS FITNESS app. Use `primary` for main CTAs (gold gradient), `secondary` for outline actions, `danger` for destructive or alert actions, `ghost` for low-emphasis.

```jsx
<Button variant="primary" size="lg">ЗАПИСАТЬСЯ</Button>
<Button variant="secondary" size="md">Подробнее</Button>
<Button variant="danger" size="md">ПРОДЛИТЬ АБОНЕМЕНТ</Button>
<Button variant="ghost" size="sm" disabled>Недоступно</Button>
```

**Variants:** primary (gold gradient + glow), secondary (gold outline), danger (red gradient), ghost (white outline)  
**Sizes:** sm (40px), md (52px), lg (60px)  
**States:** default, pressed (scale 0.97 + stronger glow), disabled (muted gray)
